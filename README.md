# Student Management System (Python)

## Project Description

This is a menu-driven Student Management System developed using Python. It allows users to manage student records efficiently.

## Features

* Add student
* View all students
* Search student
* Update student
* Delete student
* Class statistics (average, highest, lowest, topper)

## Technologies Used

* Python
* JSON (for file storage)

## How to Run

1. Install Python
2. Run the file:

   ```
   python student_management system.py
   ```

## Concepts Used

* Variables
* Lists & Dictionaries
* Functions
* Loops
* Conditions
* File Handling
